### 属性
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| type     | 类型   | string    |   header-main  /  header-main-footer  /  aisde-main  /  main-aside |     —    |
| header-height     | header高度   | string  /  number    | — |  60  |
| footer-height     | footer高度   | string  /  number    | — |  60  |
| aside-width     | aside宽度   | string  /  number    | — |  200  |
